About This Project

An inclusive tool that converts speech into sign language animations to bridge communication gaps.

Objectives

  • Assist individuals who are deaf or hard of hearing.
  • Support inclusive communication using technology.
  • Promote awareness and use of sign language.

Technologies Used

  • Flask (Python) – for backend logic
  • HTML, CSS, JavaScript – for frontend
  • Speech Recognition API – for audio-to-text conversion
  • Custom Sign Language GIFs – for visual output
