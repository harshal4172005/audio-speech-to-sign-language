About This Project

Audio Speech to Sign Language Converter is a web application built to help bridge communication gaps between hearing and non-hearing individuals. It automatically converts spoken audio into sign language animations, promoting inclusivity and accessibility.

Objectives

  • Assist individuals who are deaf or hard of hearing.
  • Support inclusive communication with assistive technology.
  • Encourage awareness of sign language in daily interactions.

Technologies Used

  • Flask (Python) – for backend routing and server logic
  • HTML, CSS, JavaScript – for interactive frontend design
  • Speech Recognition API – for converting voice to text
  • Custom Sign Language GIFs – for visual animation

Developer

Developed by Harshal Parekh, a passionate Computer Engineering student from CKPCET under Gujarat Technological University. This project reflects a vision of inclusive and accessible technology.
